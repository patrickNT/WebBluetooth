Background

This demo illustrates the use of the Network Information API to retrieve the current network type and monitor for network type changes.

The Network Information API is not currently available on all platforms. For best results, try this demo using Chrome on Android, or ChromeOS.

While the Network Information API was initially added to Chrome 38, this sample illustrates additional functionality that was added in Chrome 48. The updates include a new connection.downlinkMax attribute, support for the 'wimax' connection type, and connection.onchange event listener. The older connection.ontypechange event listener will be deprecated in a future release.
